# Linkzz
A software that generates qr codes. Will include more features soon.

## Features:
* Generates QR Codes and the image can be saved.
* Can switch theme.
* No need of internet connection.

## Future Ideas:
* Will add a URL Shortner.
* Will add custom theming for QR code.
